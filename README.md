# Amazon Clon

Este proyecto es un clon funcional de una tienda tipo Amazon desarrollado con HTML, CSS y JavaScript puro.
La aplicación renderiza dinámicamente productos, permite agregar y eliminar artículos del carrito y gestiona cantidades utilizando JavaScript.

El proyecto está hecho a partir de un curso de javasCript de mas de 22 horas de duracion con conceptos fundamentales como:

Manipulación del DOM

Modularización con import / export

Manejo de arreglos y objetos

Separación de lógica y presentación

Simulación de un carrito de compras

Arquitectura MVC.

Tecnologías usadas:

HTML5

CSS3

JavaScript
